是否在 Windows 家长控制中寻找 Web 筛选和活动报告？

您可能已注意到，Windows 7 的家长控制中并未包括 Web 限制和活动报告。 但是，通过在 Windows 家长控制中添加服务提供程序（例如 Windows Live Family Safety），您仍然可以限制孩子能够访问的站点，并查看其在线活动报告。 借助 Family Safety，您可以针对孩子设置 Web 筛选和活动监视，并决定孩子可以在 Windows Live Spaces、Messenger 和 Hotmail 中与谁通信。
