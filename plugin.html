通用安装方法

通过客户端进行安装

你可以直接在电脑客户端里点击“菜单”→ ”工具”→ “选项”→“浏览器插件”，你将看到如下界面。找到你当前使用的浏览器，并点击“启用”，然后按照提示操作，便能够安装你说需要的浏览器插件。当你安装好插件后，在你需要保存的网页上，邮件点击“保存到为知笔记”，按照提示，就可以轻松保存你想要的内容了。



通过浏览器安装

如果您的电脑没有安装客户端，或者以上列表中没有您所使用的浏览器，或者由于GFW防火墙的问题而导致一些浏览器插件按照以上方法无法成功下载，这时就需要您通过浏览器进行在线安装，或自行下载插件然后手动进行安装。

猎豹浏览器基于Chrome内核，可以通过下方的手动安装方式安装网页剪辑器

Chrome浏览器

方法1 通过应用商店安装

chrome浏览器在线安装地址：点击这里转到插件下载地址

注意：

如果网速比较慢，可能需要搜索 Wiz 或者 为知，才可以找到为知笔记网页剪辑器。

如果没有登录Google账号，需要先使用Chrome登录Google账号才可以安装Chrome应用。


方法2 手动安装

1. 下载剪辑器压缩包:下载

2. 解压缩文件到非中文目录下，例：D:\WizClipper\webclipper

3. 在浏览器中打开chrome://extensions/

4. 选中页面右上方的“开发者模式”

5. 点击“载入正在开发的扩展程序“， 找到上面解压的目录后，点击确定即可

注意：加载完扩展之后，已经打开的页面需要手动刷新一下才可以使用


使用

点击此按钮，输入为知笔记账号和密码，即可进入剪辑界面。若无为知账号，可以“创建为知笔记账号”。

进入后可以看到这样个界面

标题：默认为网页标题，可以自己修改
文件夹：默认为“我的笔记”，可以展开修改
评论：可以添加评论
下拉框：保存到本地，保存到云端（服务器）无需安装客户端即可保存到自己的为知空间中
保存整页处下拉菜单：有多种保存方式，如保存整页、保存文章、保存选择部分，保存URL等，保存到本地的时候还有更多保存方式（将会弹出保存对话框）

注意：

如果需要使用保存到本地功能，您必须安装最新版本的客户端：
最新版本客户端下载地址： http://pan.baidu.com/share/link?shareid=56903&uk=839142433

特性：

  1. 无需安装客户端即可使用，在线保存，直接保存到云端。安装客户端之后，也可以直接保存到本地。
  2. 支持剪辑模式的选择
  3. 支持windows、mac、linux
  4. 支持选择文件夹
  5. 支持对文章添加评论
  6. 支持文章内容的智能定位预览

直接保存到本地

在网页里面点击右键，在菜单中选择 选择 为知笔记网页剪辑器 -> 更多，就会出现保存网页对话框，可以选择更多地保存功能（保存到本地）

Opera浏览器

Wiz提供Opera网页发送到Wiz的接口

支持Opera版本

  1. Opera 9.x / Opera 10.x
  2. 紫电版Opera10

1、安装

Wiz安装时，会在Opera菜单上增加启动方式。如果增加失败或新装Opera，请执行菜单中的“Opera配置工具“按以下界面设置

选择Opera安装目录，点击安装

在Win7下会安装失败，请使用管理员身份运行“Opera配置工具”


2、使用

使用时，在网页右键，选择保存到Wiz

3、特殊版本Opera的手工设置

对于一些特殊版本的Opera，需要自己手工增加菜单项

标准版Opera10菜单位置：程序目录\ui\standard_menu.ini
在[Document Popup Menu]里增加：
Item, "保存到WizKnowledge"=Execute program, "C:\Windows\system32\rundll32.exe", "C:\PROGRA~1\WIZBRO~1\WIZKNO~1\WIZKMC~1.DLL, WizKMURLDownloadToDocument /ShowProgress=1 /URL=%u /CacheFile=%s"


紫电版Opera10 菜单位置：程序目录\profile\menu\Menu.ini
在[Document Popup Menu]里增加：
Item, "保存到WizKnowledge"=Execute program, "C:\Windows\system32\rundll32.exe", "C:\PROGRA~1\WIZBRO~1\WIZKNO~1\WIZKMC~1.DLL, WizKMURLDownloadToDocument /ShowProgress=1 /URL=%u /CacheFile=%s"


白菜版OperaEz菜单位置：corals_lite_menu.ini 增加：
Item, "保存到WizKnowledge"=Execute program, "C:\Windows\system32\rundll32.exe", "C:\PROGRA~1\WIZBRO~1\WIZKNO~1\WIZKMC~1.DLL, WizKMURLDownloadToDocument /ShowProgress=1 /URL=%u /CacheFile=%s"


4、注意

使用Opera10时，请使用10.52及以上版本。10.50和10.51版本opera在右键“保存到Wiz”时无法调用程序
