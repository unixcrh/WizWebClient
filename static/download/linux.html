Wiz for ubuntu安装指南

下载与平台对应的安装包：

bin

  • 32位: wiznote-1.0-beta-i386.tar.gz
  • 64位: wiznote-1.0-beta-amd64.tar.gz

Ubuntu安装包

  • 32位: wiznote-1.0-beta-i386.deb
  • 64位: wiznote-1.0-beta-amd64.deb

Wiz正确的运行需要依赖以下软件包：

$ apt-cache depends wiznote
wiznote
Depends: libc6
Depends: libgcc1
Depends: libqt4-network
Depends: libqt4-xml
Depends: libqtcore4
Depends: libqtgui4
Depends: libqtwebkit4
Depends: libstdc++6

安装平台：ubuntu 11.10 amd64/i386
11.10发行版已经内置了大多数依赖，除了webkit包：

$ sudo apt-get install libqtwebkit4

然后执行安装：

$ sudo dpkg -i wiznote_1.0-1_i386.deb

或者

$ sudo dpkg -i wiznote_1.0-1_amd64.deb

安装完成之后，可以在Unity中找到wiznote启动图标

安装平台：ubuntu 10.04 amd64/i386

因为10.04的软件仓库是没有libqtwebkit4依赖包的，所以首先需要开启backport更新支持，关于什么是backport，英文好的用户可以参考一下这里：

https://help.ubuntu.com/community/UbuntuBackports#Enabling_Backports

这里我们采用命令的方式开启backport支持：

$ sudo add-apt-repository ppa:kubuntu-ppa/backports

更新一下本地软件包列表：

$ sudo apt-get update

因为在10.04不像11.10，大部分依赖都默认安装了，所以我们需要手动安装：

$ sudo apt-get install libqt4-network libqt4-xml libqtcore4 libqtgui4 libqtwebkit4

最后执行安装：

$ sudo dpkg -i wiznote_1.0-1_i386.deb

或者

$ sudo dpkg -i wiznote_1.0-1_amd64.deb

安装完成之后，在Applications->Accessories下找到wiznote启动图标

Version of libqtwebkit4 on system is 2.0.0-0ubuntu1~lucid1~ppa1.
